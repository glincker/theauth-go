# TheAuth — Go

> The auth library for AI agents and the humans who run them.

Session-based auth for Go with magic links, OAuth (coming v0.2), and MCP OAuth 2.1 (coming v2.0).

🚧 Pre-release. v0.1 ships magic-link auth, sessions, chi middleware, Postgres + in-memory storage.

See [theauth.dev](https://theauth.dev) (coming soon).

Sibling project: [theauth (TypeScript)](https://github.com/glincker/theauth).
